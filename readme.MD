# Goweb ![](https://github.com/twharmon/goweb/workflows/Test/badge.svg) [![](https://goreportcard.com/badge/github.com/twharmon/goweb)](https://goreportcard.com/report/github.com/twharmon/goweb) [![](https://gocover.io/_badge/github.com/twharmon/goweb)](https://gocover.io/github.com/twharmon/goweb)
Light weight web framework based on net/http.

Includes
- routing
- middleware
- autoTLS
- logging

## Usage
See [examples](https://github.com/twharmon/goweb/tree/master/examples).

## Documentation
For full documentation see [godoc](https://godoc.org/github.com/twharmon/goweb).
