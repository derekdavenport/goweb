# Goweb ![](https://github.com/twharmon/goweb/workflows/Test/badge.svg)
Light weight web framework based on net/http.

Includes
- routing
- middleware
- autoTLS
- logging

See [examples](https://github.com/twharmon/goweb/tree/master/examples).
